About the Project

This project explores archaeological datasets to analyze artifact distribution across different sites. Using Python for data preparation and Tableau/Flourish for visualization, the aim is to uncover patterns that help identify historically significant locations.

Methods

  • Data Cleaning & Processing (Python, Pandas)
  • Visualization & Charts (Tableau)
  • Dashboard Creation (5-sheet professional view & 10-sheet complete view)

Expected Outcome

The project provides archaeologists with clear insights into artifact distribution. By comparing professional (5-sheet) and complete (10-sheet) dashboards, decision-makers can focus on key findings while also having access to detailed data views.
